# Project 001 - Cycle Counter

## 📌 Overview

This project implements a simple cycle counter using Arduino.

The goal is to build the basic logic of an industrial pad testing machine.

---

## Features

- Button Input
- LED Output
- Cycle Counter
- Serial Monitor Output
- Test Complete Message

---

## Hardware

- Arduino UNO
- Push Button
- LED
- 220Ω Resistor
- Breadboard

---

## Pin Configuration

| Device | Pin |
|---------|-----|
| Button | D7 |
| LED | D8 |
